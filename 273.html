Ejercicio No 273


You have the following CSS markup that renders a red rectangle on a webpage.


How large will the red rectangle appear when it renders on the page?

A. width 200px, height 150 px
B. width 400px, height 350 px
C. width 100px, height 50 px
D. width 300px, height 250 px

Correct Answer: A

Section: Volume D

Explanation

Explanation/Reference:

References: https://www.w3schools.com/CSSref/css3_pr_box-sizing.asp



Explicación

1. Cuando aplicamos box-sizing: border-box: El ancho y el alto se aplican a todas las partes del elemento: contenido, relleno y bordes: Aquí, en nuestro ejercicio el ancho completo es de 200px, ¡pase lo que pase!

2. Si usamos (que no es el caso) box-sizing: content-box (default): El ancho y la altura solo se aplican al contenido del elemento: Si aplicamos esta propiedad entonces el div tiene un ancho de 200px. Pero el ancho completo es 200px + 100px (margin izquierdo y derecho) + 100px (padding izquierdo y derecho) = 400px!


Respuesta:


  • Opción A: usando width 200px, height 150 px


  • Opción B: usando width 400px, height 350 px


  • Opción C: usando width 100px, height 50 px


  • Opción D: usando width 300px, height 250 px
