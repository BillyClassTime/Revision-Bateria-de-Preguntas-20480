You have the following markup.




You need to ensure that flex1 is displayed on the right of the page. flex2 must be displayed directly to the left of flex1.

Which style should you add to the outer div?



A. Option A
B. Option B
C. Option C
D. Option D

Correct Answer: B

Section: Volume D

Explanation

Explanation/Reference:

Reference: https://www.w3schools.com/cssref/css3_pr_flex-direction.asp

Explicación


El ejercicio requiere que nosotros invertamos la forma en que se muestran los div. Sin aplicar el estilo los divs se muestran como sigue a continuación:

Flex1
Flex2

La idea es que el div rojo aparezca primero que el div amarrillo. Para ello debemos agregar el estilo apropiado al div externo.

Respuesta:
Opción A, usando el estilo: flex-direction:column-reverse; Como se observa los div se situan uno sobre el otro y no es lo que se quiere.

Flex2


La respuesta correcta es la opción B: flex-direction:row-reverse; Que nos ayuda a invertir el orden de los div

Flex2


Opción C, usando el estilo: flex-direction:row; Aquí se mantiene lo mismo que fue dado inicialmente y no cumplimos con lo requerido.

Opción D, usando el estilo: flex-direction:column. Aquí invertimos el orden de las columnas pero no cumple con lo querido.
