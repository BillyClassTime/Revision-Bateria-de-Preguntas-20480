QUESTION 35 You are styling a box object on a page by using CSS3. You need to set the transparency of the object to 50%. Which two CSS3 styles will achieve the goal? (Each correct answer presents a complete solution. Choose two.) A <style> .Box { background: rgb (255, 0, 0); backface-visibility: hidden; /*Visibilidad de la cara posterio: define si la cara posterior de un elemento debe ser visible cuando se enfrenta a un ususio */ } </style> B <style> .Box { background: rgb (255, 0, 0); background: rgba (255, 0, 0, .5); } </style> C <style> .Box { background: rgb (255, 0, 0); opacity: 0, 5; } </style> D <style> .Box { background: rgb (255, 0, 0); background-color: #fff; /*Background is the Shorthand background-color*/ } </style> Correct Answer: BC Section: Volume A Explanation Explanation/Reference: Explanation: The RGBA declaration allows you to set opacity (via the Alpha channel) as part of the color value. Example: <style> div { background-color: rgba(255, 0, 0, 0.5); } </style> The background color has been set to blue, and the opacity set to half. Example 2: <style> div { background-color: rgb(0, 0, 255); opacity: 0.5; } </style> The background color has been set to blue, and the opacity set to half.
