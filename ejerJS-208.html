Ejercicio No 208


You need to parse an XML formatted string. Which object should you use?

A. Map
B. DataView
C. DOMParser
D. JSON

Correct Answer: C

Section: Volume D

Explanation/Reference:



Explicación

Esto es solo una pregunta teórica. Todos los navegadores principales tienen un analizador XML incorporado para acceder y manipular XML.

El DOM XML (Document Object Model) define las propiedades y métodos para acceder y editar XML.

Sin embargo, antes de poder acceder a un documento XML, debe cargarse en un objeto DOM XML.

Todos los navegadores modernos tienen un analizador XML integrado que puede convertir texto en un objeto DOM XML.



Respuesta:

Por lo tanto la respuesta correcta es la D. A continuación una respuesta gráfica y con código de lo antes expuesto.

Lo primero que haremos es crear un parráfo que vamos a cargar de forma dinámica. Para ello haremos uso de un script. En este definimos un texto XML el cual lo recuperamos con una variable llamada "text".
Seguidamente instanciamos un analizador llamado "parser" del DOMparser(), es decir creamos un objeto analizador del DOM.
Este objeto se utiliza para definir el tipo de texto que vamos a analizar "text/xml" y al mismo tiempo le pasamos el texto a analizar que es nuestro texto XML "text".
El resultado de lo anterior se asigna a una nueva variable "xmlDoc".
Esta última se utiliza para llegar hasta la rama del XML que queremos que en este caso es "title". Lo cual se le asigna al documento utilizando "getElementById("demo") para pintarlo en el HTML por medio de "innerHTML".

Todo lo expuesto se muestra aquí abajo.

<p>id="demo"></p>

<script>

var text, parser, xmlDoc;

text = "<bookstore><book>" +

"<title>Everyday Italian</title>" +

"<author>Giada De Laurentiis</author>" +

"<year>2005</year>" +

"</book></bookstore>";

parser = new DOMParser();

xmlDoc = parser.parseFromString(text,"text/xml");

document.getElementById("demo").innerHTML =

xmlDoc.getElementsByTagName("title")[0].childNodes[0].nodeValue;

</script>
