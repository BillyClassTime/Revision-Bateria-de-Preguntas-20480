You need to write a self-executing anonymous function in JavaScript. Which code snippet should you include in the function?

B. Option B

Explicación

El arte de la función anónima autoejecutable.

(function(){
console.log('¡Hola mundo!');
})();

Al observar el código detenidamente veremos que se compone de dos partes clave.

    1. Primero está la función anónima:

     (función(){
    // El código normal va aquí
    })

     2. La parte realmente interesante es lo que sucede cuando agregamos esto justo al final:

     ();
    Esos dos pequeños paréntesis hacen que todo lo contenido en las llaves anteriores se ejecute inmediatamente.
Lo que es útil aquí es que JavaScript tiene un alcance a nivel de función. Todas las variables y funciones definidas dentro de la función anónima no están disponibles para el código fuera de ella, utilizando el cierre de manera efectiva para aislarse del mundo exterior.



Respuesta:

Por lo expuesto anteriormente vemos primero que la función es anónima por que no lleva nombre. Esto elimina las opcionces B y C.

Para que la función se auto ejecute tiene que llevar la sintaxis de ejecución por ejemplos: function (); es decir que los paréntesis al final tienen que aparecer. Esto elimina la opción A.

Por lo tanto la respuesta correcta es la opcion D.
