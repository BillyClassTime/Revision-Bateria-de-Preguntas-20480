QUESTION 306

You have the following HTML Markup.
You need to ensure that the markup conforms to W3C accessibility standards.
What should you add to the markup?
<figure> <img src="/images/ourheadquartes.jpg" > </figure>
  1. A. a figcaption element within the figure element
  2. B. an alt attribute to the img element
  3. C. a name attribute to the img element
  4. D. a header element within the figure element

Correct Answer: B

Explanation:
Images should include alternative text (alt text) in the markup
References:
https://www.w3.org/standards/webdesign/accessibility
https://www.w3.org/TR/UNDERSTANDING-WCAG20/text-equiv.html
